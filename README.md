# 🧪 Virtual Enzyme Kinetics Lab

This project is a Streamlit-based virtual lab that simulates enzyme kinetics using the Michaelis-Menten equation.

## 🚀 Features
- Interactive sliders for Vmax and Km
- Real-time graph visualization
- Educational explanation

## ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
